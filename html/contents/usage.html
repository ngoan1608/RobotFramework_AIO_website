Usage

Installation

The installation process is incredibly straightforward, as all the necessary components are conveniently packaged in the installer file above. Additionally, you can complete the installation offline, without needing an internet connection.

  • Windows

    1. Download the installer package for Windows.

    2. Run the installer and follow the on-screen instructions to complete the installation process.

    3. Once the installation is completed, launch the VSCodium for RobotFramework from the Start menu.

  • Linux

    1. Download the installer package for Linux.

    2. Open a terminal and navigate to the directory where the downloaded file is located.

    3. Run the following command to make the file executable:

    chmod +x RobotFramework_AIO_Linux_<n.n.n.n>.deb

    4. Install the package using the following command:

    sudo dpkg -i RobotFramework_AIO_Linux_<n.n.n.n>.deb

    5. Once the installation is completed, launch the VSCodium for RobotFramework from the applications menu.
