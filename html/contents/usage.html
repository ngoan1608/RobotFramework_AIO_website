Usage

Installation

The installation is very simple because all required stuffs is packaged in the above installer files. Besides, the installation can be perform offline without internet.

  • Windows

    1. Download the installer package for Windows.

    2. Run the installer and follow the on-screen instructions to complete the installation process.

    3. Once the installation is completed, launch the VSCodium for RobotFramework from the Start menu.

  • Linux

    1. Download the installer package for Linux.

    2. Open a terminal and navigate to the directory where the downloaded file is located.

    3. Run the following command to make the file executable:

    chmod +x RobotFramework_AIO_0.6.0.5-0_amd64.deb

    4. Install the package using the following command:

    sudo dpkg -i RobotFramework_AIO_0.6.0.5-0_amd64.deb

    5. Once the installation is completed, launch the VSCodium for RobotFramework from the applications menu.

Execute RobotFramework testcase

The VSCodium Editor which is packaged in RobotFramework AIO is pre-installed the extensions and pre-configured the required settings, so that you do not need to install or configure anything, just click Run or Debug button to execute the RobotFramework testcase.

Besides, the RobotFramework test file *.robot can be executed directly on Command Prompt (Windows) and Terminal (Linux)

The Tutorial testcase folder

User can get familiar with RobotFramework easily with the testcases defined in the Tutorial folder.

This folder is already added in the VSCodium workspace, so that you can run them directly to get the result.
