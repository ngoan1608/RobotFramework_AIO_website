Features

An All-In-One package which contains all required packages, tools and libraries in one installer file

  • Python Interpreter - version 3.9
  • Robot Framework core - branch of version 6.02 with extensions (details below)
  • VSCodium Editor with pre-installed helpful extensions and pre-configured workspace
  • Additional Python and Robot Framework libraries

Robot Framework core extensions

We extend the Robot Framework core according to our requirements. Goal is to bring this changes into the community version that we can use a standard Robot Framework core.

  • Threading (page 13, chapter 7)
    This feature is currently in stabilization
  • Loglevel "USER" (page 12, chapter 6)
    For this feature we created already a pull-request for the Robot Framework core

VSCodium Editor with pre-installed extensions, pre-configured workspace and the tutorial Robot test folder

  • Pre-installed Python, Robot Framework Language Server and other useful extensions
  • Pre-configured workspace and extensions so that you just click Run/Debug button to execute Robot testcase(s)
  • The tutorial test folder for you to practice with Robot Framework

RobotFramework_TestsuitesManagement

RobotFramework AIO offers scalable management of an unlimited number of projects and variants through JSONP. JSONP is an extended version of JSON that allows for comments, imports, and variables at the JSON level, making it incredibly flexible and powerful for managing your test suites.

  • Scalable Management: Easily manage numerous projects and variants.
  • Advanced JSONP: Utilize comments, imports, and variables directly in JSON.
  • Flexible Configuration: Adapt and configure your test suites effortlessly.

RobotFramework AIO package provides the solution to import the Robot Framework result file(s) (output.xml), PyTest result file(s) (as JUnitXML format) to the Test Management applications:

  1. IBM Engineering Test Management: (from version 7.0.0) which is as know as Rational Quality Manager - RQM (for the older versions).
    • RobotLog2RQM tool
  2. TestResultWebApp: A web application for managing test execution results systematically and processing, displaying of test result data in a sophisticated way.
    • RobotLog2DB tool
    • PyTestLog2DB tool
