Features

Lists of RobotFramework AIO's features

  • All required packages and libraries in one installer file
  • Packaged Python 3.9 with required dependencies and helpful packages
  • Packaged Robotframework with extended libraries
  • VScodium IDE with required extensions
  • Variants and json configuration files with RobotFramework_TestsuitesManagement
  • Mechanism to handle trace log continously receiving from a connections with QConnectBase
  • Import tools to process the Robotframework/PyTest results
  • Testresult Webapp infrastructure
